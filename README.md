# Secure CI/CD Pipeline for a Web Application

This project is a hands-on DevSecOps lab focused on integrating security checks into a CI/CD pipeline.

The goal is to detect security issues early in the development workflow using automated tools such as SAST, secret detection, dependency scanning and container image scanning.

## Planned Security Checks

- Static Application Security Testing with Semgrep
- Secret detection with Gitleaks
- Dependency and filesystem vulnerability scanning with Trivy
- Docker image vulnerability scanning with Trivy
- Basic DAST scan with OWASP ZAP

## Tech Stack

- Node.js
- Express.js
- Docker
- GitHub Actions
- Semgrep
- Gitleaks
- Trivy
- OWASP ZAP
